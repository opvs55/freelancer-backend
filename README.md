## Sobre o projeto

Este projeto foi desenvolvido pensando em criar uma ponte entre empresas e trabalhadores freelancers, surgiu como forma de prática de reforço das aprendizagens do bootcamp labenu.


## Tecnologias


- NodeJS
- Typescript
- Express
- SQLite3
- KNEX
- POSTMAN
- ROUTER
- POO
